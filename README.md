# Blow It Up - Physics Sandbox

An interactive 2D physics simulator developed in C++, using Box2D for force calculations and SFML for graphical rendering.

## 🚀 Features
- **Physics Simulation:** Realistic interactions between objects (boxes, beams, balls).
- **Destruction System:** Objects fragment into debris (`shatter`) under the impact of explosions.
- **Diverse Arsenal:** Grenades and nuclear bombs with vector-calculated shockwaves.
- **Particle System:** Visual effects for explosions, smoke, and sparks.

## 🛠️ Technologies Used
- **Language:** C++ 20
- **Physics Engine:** [Box2D](https://box2d.org/)
- **Graphical Interface:** [SFML 3.0](https://www.sfml-dev.org/)

## 🎮 How to Play
1. **Build:** Select tools from the top bar to place objects.
2. **Destroy:** Place explosives and press `SPACE` to detonate.
3. **Reset:** Press `R` to clear the scene.
